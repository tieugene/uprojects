Human Interface Guide

Пусть будет Person.detail:
* запись со своими атрибутами:
* атрибуты - в табличку Название: Значение
* с подмоделями:
* все значения подмоделей - в строку каждое.
* всё редактируется inplace, без перегрузки страницы
Редактируем
[ Delete ]
Main:
    * [ Edit ]  =>
    * [ Save ] [ Cancel ]
    ! restore on error (mark errors w/ red bg)
Subs:
    * [ Edit ] [ Delete ] =>
    * [ Save ] [ Cancel ]
